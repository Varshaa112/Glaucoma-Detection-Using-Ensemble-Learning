# Ensemble Learning with Explainable AI for Accurate Glaucoma Detection Using Fundus Images

This project focuses on detecting glaucoma using deep learning and Explainable AI techniques on retinal fundus images. The preprocessing pipeline includes resizing images to 224x224, green channel extraction, CLAHE, normalization, and data augmentation.

Pretrained CNN models such as ResNet50 and EfficientNetB3 are used for classification, and ensemble learning techniques are applied to improve accuracy and robustness. Explainable AI methods like Grad-CAM and LIME are integrated to make predictions more interpretable.

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib

## Models Used
- ResNet50
- EfficientNetB3
- Ensemble Learning

## Explainable AI Techniques
- Grad-CAM
- LIME
